# guru-api-wrapper

##To install

Add the following to your composer.json file and run `composer update`

```

{
    "repositories":
    [
       {
         "type": "vcs",
         "url": "https://github.com/little-guru/guru-api-wrapper"
       }
    ],
    "require":
    {
       "little-guru/guru-api-wrapper" : "dev-master"
    }
}

```
